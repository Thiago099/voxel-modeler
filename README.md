# web-gl-get-face-mouse-is-over
## Install

npm install

## Run

npm run dev

## Usage

<h2>
Tips:
</h2>
<p>
<h3> Camera: </h3>
<p>
● Left click and drag to rotate
</p>
<p>
● middle click and drag to pan
</p>
<p>
● Scroll to zoom
</p>
</p>
<p>
<h3> Object:</h3>
<p>
● Left click to add a voxels from the highlighted faces
</p>
<p>
● Right click to remove a voxel from the highlighted faces
</p>
